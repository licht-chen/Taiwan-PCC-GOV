# Taiwan-PCC-GOV
## Indroduction

針對台灣政府採購網 「招標公告、公開徵求、公開閱覽」三大招標類型 設計自動化爬蟲 以及 GUI 介面，可搜尋期望的機關名稱(ex: 中華郵政)，在特定公告日期內的基本招標資訊，以提供相關使用者初步的資料蒐集

## Features

### PCC_GOV_GUI
GUI 介面

### PCC_GOV_search()
針對 招標公告 進行爬蟲

### PCC_GOV_searchAppealVendor()
針對 公開徵求 進行爬蟲

### PCC_GOV_QueryPublicReadData()
針對 公開閱覽 進行爬蟲

### PCC_GOV_get_orgname
針對 機關名稱 進行爬蟲
